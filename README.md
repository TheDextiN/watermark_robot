# Telegram Robot: Add watermark to photo

Dependencies:
```
python v3.x
pip
```

Install requirements:
``` bash
$ sudo pip3 install -r requirements.txt
```

! Edit api in script.

Usage:
``` bash
$ python3 app.py
```

``` bash
# docker build -t watermark .
```
